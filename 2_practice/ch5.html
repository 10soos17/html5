1. 관 련 있는 것끼리 연결하시오.

(1) 기본 선택자
①-ⓒ 전체선택자, *
②-ⓓ 태그선택자, h1
③-ⓑ 아이디선택자, #id
④-ⓐ 클래스선택자, .header
(2) 자손 선택자와 후손 선택자
①-ⓐ 선택자A>선택자B, 자손선택자
②-ⓑ 선택자A 선택자B, 후손선택자

2. 다 음 질문에 OX를 표시하시오.

(1) 자손은 후손보다 넓은 개념입니다. ( x )
(2) div > h1, h2는 “div 태그 아래”의 “h1 태그와 h2 태그”를 선택하는 선택자입니다. ( x )
>(3) CSS도 HTML처럼 형태로 주석을 사용합니다. ( x )
>(4) 2em은 2배를 나타내는 크기 단위입니다. ( o )
(5) 색상 단위에서 투명도를 지정할 때는 argb() 등을 사용합니다. ( x )
(6) URL을 지정할 때는 url() 형태의 코드를 사용합니다. ( o )

3. 다 음 코드에서 선택자가 어떤 태그를 선택하는지 설명하시오.

(1) header a (header 태그의 후손 중에 a 태그를 선택합니다.)
(2) header > * > h1 (header 태그의 모든 자손들을 선택 후, 그 자손이 h1인 태그를 선택합니다.)
(3) header > div:first-child > a (header 태그의 자손 중에 div 태그를 선택한 후, div 태그의 형제들 중에 첫번째를 선택한 후, 그 자손인 a태그를 선택합니다.)
(4) header a:first-child (header 태그의 후손 중에 a 태그를 선택한 후, a 태그의 형제들 중에 첫 번째를 선택합니다.)
(5) div > h1, h2 (div 태그의 자손 중에 h1과 h2인 태그를 선택합니다.)
(6) .title.card (클래스 이름이 title card라는 클래스를 선택합니다.)
