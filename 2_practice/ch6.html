(5) background-( image ) 속성은 배경 이미지를 넣을 때 사용합니다.

(18) ( position ) 속성은 요소의 위치 지정 형식을 설정할 때 사용합니다.
(19) ( overflow ) 속성은 내용이 요소 크기를 벗어나 모두 보여 주기 힘들 때 어떻게 보여 줄지를 지정합니다.
(20) ( border-shadow ) 속성은 박스에 그림자를 부여합니다.
(21) ( text-shadow ) 속성은 글자에 그림자를 부여합니다.

(6) 일반적으로 웹 브라우저의 글자 기본 크기는 16px입니다. ( o )
